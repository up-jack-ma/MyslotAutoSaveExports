# MyslotAutoSave

一个**独立运行**的魔兽世界插件，用于在登录时自动备份角色的动作条、按键绑定和宏设置。
它与 [Myslot](https://www.curseforge.com/wow/addons/myslot) 完美兼容，备份的数据会自动出现在 Myslot 的"导入"列表中。

## 主要功能

*   **自动备份**: 每次角色进入游戏世界（登录或重载界面）时，自动执行备份。
*   **智能命名**: 备份名称格式为 `角色名-专精-服务器` (例如: `PlayerName-Frost-Dalaran`)。
*   **Myslot 集成**: 备份数据直接写入 Myslot 的存储 (`MyslotExports`)，你可以直接在 Myslot 插件的界面中找到并恢复这些备份。
*   **防重复**: 如果当前角色/专精的当天备份已存在，会自动跳过，避免刷屏。
*   **完全独立**: 由于魔兽插件的隔离机制，本插件内置了所有必要的导出逻辑和库文件，**不需要**依赖 Myslot 插件即可运行（但为了恢复数据，你需要安装 Myslot）。

## 为什么需要这个插件？

Myslot 是一个非常强大的插件，但它没有提供"自动定时备份"的功能。
由于魔兽世界插件环境的限制（Sandbox 隔离），我们无法直接调用 Myslot 内部的函数来执行导出。
因此，本插件完整复刻了 Myslot 的导出逻辑，作为一个轻量级的"自动触发器"，默默地帮你守护配置安全。

## 安装方法

1.  下载插件包。
2.  解压并放入 `_retail_/Interface/AddOns/MyslotAutoSave` 目录。
3.  确保你同时也安装了 **Myslot** (用于以后恢复数据)。
4.  进入游戏，你会看到聊天框提示 `[MyslotAutoSave] 自动备份成功: ...`。

## 如何恢复备份？

1.  打开 Myslot 插件的主界面 (`/myslot`)。
2.  点击 **"导入" (Import)** 按钮。
3.  在下拉菜单中，你就能看到本插件自动生成的备份记录了。
